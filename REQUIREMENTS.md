# 🐱 鱼仔桌面宠物 — 需求文档

> **项目名称：** 鱼仔桌面宠物 (YuZai Desktop Pet)  
> **目标平台：** Windows 10+ / macOS 12+（通过 Electron 跨平台）  
> **技术栈：** Electron + TypeScript + Canvas 2D + esbuild  
> **当前版本：** v1.0.0（早期开发中）

---

## 一、项目背景

鱼仔是你家里养的一只猫 🐱。现在希望让鱼仔'住进'你的电脑桌面，成为一只可爱的桌面宠物——在你工作时陪伴你，在你的屏幕上走动、睡觉、追逐鼠标、发出呼噜声，成为你的桌面小伙伴。

### 2.2 当前猫的行为状态

```
idle      → 发呆（随机走动 / 睡觉）
walking   → 屏幕内随机漫步
sleeping  → 睡觉（打呼噜 Zzz、呼吸动画）
chasing   → 追逐鼠标光标
dragged   → 被鼠标拖拽中
purring   → 鼠标悬停时发出呼噜声
eating    → 吃食物（暂为占位动画）
meowing   → 点击后喵叫
```

- [ ] 代码注释完善（中文注释）

### 4.3 渲染方案升级
当前为 Canvas 2D 程序化绘制。建议升级方案：

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **精灵图 (Sprite Sheet)** | 性能好、实现简单、照片级外观 | 需要美术资源、状态多了图会变大 | ⭐⭐⭐⭐⭐ |
| **Lottie / Rive 动画** | 矢量流畅、文件小 | 学习成本、工具链依赖 | ⭐⭐⭐ |
| **GIF/APNG** | 极简实现 | 背景透明问题、文件大 | ⭐⭐ |
| **骨骼动画 (Spine)** | 可组合、复用高 | 复杂度高、需要专业工具 | ⭐⭐ |

**建议：** 使用精灵图方案。将鱼仔的照片按状态排成 spritesheet，Canvas 按帧绘制即可。

### 4.4 性能要求
- 空闲态 CPU 占用 < 1%
- 动画帧率稳定 60fps
- 内存占用 < 100MB
- 启动时间 < 2秒

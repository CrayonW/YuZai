# 🔊 鱼仔音频素材

在这里放置鱼仔的叫声和其他音频文件。

## 建议放置内容

| 文件名 | 内容 | 说明 |
|--------|------|------|
| `meow.mp3` | 🐱 鱼仔叫声 | 录一段鱼仔的真实叫声，短促为宜（1-2秒） |
| `meow-long.mp3` | 😸 长叫 | 鱼仔拉长音的叫声（撒娇/要吃的） |
| `purr.mp3` | 😌 呼噜声 | 鱼仔被摸时的呼噜声，建议循环播放（5-10秒） |
| `eat.mp3` | 🍽️ 吃东西声 | 猫粮咀嚼/舔食声音 |

## 注意事项

- 格式建议：MP3（兼容性好）
- 呼噜声建议录一段连续的，代码中会循环播放
- 文件尽量小（<500KB 每个）
- 如果没有真实录音，可以先找免费猫咪音效代替，等鱼仔录好后再替换

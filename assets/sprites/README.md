# 🎞️ 鱼仔精灵图 / 动画

在这里放置鱼仔的精灵图（spritesheet）和动画素材。

## 建议精灵图规格

建议制作一张 1024×512（或更大）的 PNG 精灵图：

```
spritesheet.png
├── 第1行: idle (4帧)    — 鱼仔站着偶尔摇尾巴
├── 第2行: walk (6帧)    — 鱼仔走路循环
├── 第3行: sleep (3帧)   — 鱼仔蜷缩睡觉
├── 第4行: meow (4帧)    — 鱼仔张嘴叫
├── 第5行: eat (4帧)     — 鱼仔吃东西
├── 第6行: happy (3帧)   — 鱼仔开心蹭
├── 第7行: dragged (2帧) — 鱼仔被拎起来
└── 第8行: grooming (4帧) — 鱼仔洗脸
```

## 备选方案

- 也可以放置 Lottie JSON 动画文件
- 或使用逐帧 PNG 序列（每帧一个文件）

> 先用 `assets/images/` 下的照片制作，完成后放入此目录。

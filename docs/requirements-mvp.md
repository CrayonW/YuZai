# 鱼仔桌面宠物 MVP 需求规格

## 当前实现范围

第一版只实现本地开发可运行的桌面宠物原型：

- 透明、无边框、置顶 Electron 窗口
- Canvas 2D 简易绘制鱼仔
- `front / left / right` 三视角
- `idle / walking / sleepy / sleeping / waking / surprised / shy / dragging / waving`
- Idle 作为全局回流中心
- 所有状态切换进入过渡计时
- 点击、连续点击、长按拖拽
- 默认鼠标穿透，宠物命中区域可交互
- 右键菜单：隐藏、显示、重置位置、动作频率、退出
- 桌面边缘 20px 回弹

## 明确不实现

- 办公软件检测
- 低电量、系统弹窗、保存成功等系统联动
- 配饰系统
- 用户自定义动作导入

## 后续允许扩展

- exe/dmg 安装包
- 天气联动动作
- 日程提醒动作
